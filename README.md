# 🚗 Driver Drowsiness Detection System

## 📌 Description
This project detects driver drowsiness in real-time using computer vision techniques. It monitors eye movements and blinking patterns to determine whether the driver is Active, Drowsy, or Sleeping.

---

## 🚀 Features
- Real-time face detection using webcam  
- Eye blink detection using facial landmarks  
- Driver state classification:
  - Active  
  - Drowsy  
  - Sleeping  
- Alert system for sleeping detection  

---

## 🛠️ Technologies Used
- Python  
- OpenCV  
- Dlib  
- NumPy  
- Imutils  

---

## ▶️ How to Run

### 1. Install Dependencies
pip install opencv-python dlib imutils numpy

### 2. Download Required File  
Download the file from:  
http://dlib.net/files/shape_predictor_68_face_landmarks.dat.bz2  

### 3. Extract File
Extract the downloaded .bz2 file to get:  
shape_predictor_68_face_landmarks.dat  

### 4. Setup
Place the .dat file in the project folder  

### 5. Run the Program
python ddd.py
